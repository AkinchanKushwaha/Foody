# Foody App
This is a food recipes android app with clean architecture made with the help of [Spoonacular API](https://spoonacular.com/food-api).

- Coroutines
- MVVM
- Dagger Hilt
- Retrofit
- Room
- Navigation Components
- Cache Strategy (Repository Pattern)
- Coroutines Flow
- ViewBinding
